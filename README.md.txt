# 🏋️ Fitness Buddy — AI Health & Fitness Coach

An intelligent conversational AI agent built using 
Langflow that provides personalized fitness guidance,
nutrition plans, and motivation.

## 🌟 Features
- 💪 Personalized workout recommendations
- 🥗 Custom nutrition and meal plans
- ⚡ AI motivation coach (SPARK Agent)
- 🔍 Real-time web search for latest trends
- 🧮 BMI and calorie calculator
- 💬 Conversational memory

## 🏗️ Architecture
- Main Fitness Agent — coordinates everything
- NUTRI Agent — nutrition and meal planning
- SPARK Agent — motivation and encouragement
- DuckDuckGo Search — real-time web search
- Calculator Tool — BMI and calorie calculations
- Message History — conversation memory

## 🛠️ Tech Stack
- Langflow
- IBM WatsonX (Meta Llama 3.3 70B)
- DuckDuckGo Search API
- Python

## 🚀 How to Run
1. Install Langflow:
pip install langflow

2. Start Langflow:
langflow run

3. Import the flow:
→ Open http://localhost:7860
→ Click Import
→ Upload FitnessBuddy.json

4. Configure credentials:
→ Add your WatsonX API key
→ Add your Project ID
→ Set URL to https://au-syd.ml.cloud.ibm.com

5. Click Playground and start chatting!
